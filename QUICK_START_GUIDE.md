2. Go to **Settings** > **Profile**
3. Change password from `@adminKMCI` to your secure password
4. Update your name from "AdminKMCI" to your actual name

### **Step 2: Add Your Church Content**
1. **Home Page Content:**
   - Upload your church logo in Settings
   - Update hero section text
   - Add welcome message

2. **About Page:**
   - Add your pastor's photo and bio
   - Update mission and vision statements
   - Add staff information

3. **Contact Information:**
   - Update church address and phone
   - Add service times
   - Set correct map location

### **Step 3: Create Your First Content**
1. **Blog Post:**
   - Admin > Blog > New Post
   - Add title, content, featured image
   - Set status to "Published"

2. **Upcoming Event:**
   - Admin > Events > New Event
   - Add event details and registration
   - Upload event image

3. **Recent Sermon:**
   - Admin > Sermons > New Sermon
   - Upload audio/video file
   - Add sermon notes and scripture
